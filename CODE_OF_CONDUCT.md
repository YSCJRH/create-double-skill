# Code of Conduct

Short version: be respectful, specific, and honest about tradeoffs.

## Our Pledge

为了给所有人提供一个尽量友好、专注、可持续的协作环境，我们承诺：

- 以尊重和善意交流
- 允许不同经验层级的人提问、反馈和贡献
- 讨论问题时聚焦行为、事实、影响和改进方向
- 不因为身份背景、经验水平、表达方式或观点不同而贬低他人

## Our Standards

有助于社区健康的行为包括：

- 给出具体、可执行、可复现的反馈
- 在批评设计或实现时保持克制和清楚
- 尊重项目边界、维护成本和隐私约束
- 接受别人可能和你有不同的优先级与取舍
- 在不确定时先问清楚，再下判断

不可接受的行为包括：

- 骚扰、羞辱、威胁、歧视或人身攻击
- 恶意挑衅、引战、持续打断或故意消耗他人
- 发布他人的私人信息、敏感数据或未获授权的内容
- 明知存在风险仍鼓励公开提交私密样本、真实个人资料或安全敏感信息
- 任何让讨论环境明显变得不安全或不可持续的行为

## Enforcement Responsibilities

这个仓库目前由个人维护。维护者会尽量：

- 审慎查看行为问题反馈
- 在必要时要求停止不当行为
- 视情况移除评论、关闭讨论、限制互动或拒绝贡献

这里不会承诺复杂的治理流程或固定处理时限，但会尽量认真对待每一份相关反馈。

## Scope

本行为准则适用于与本仓库相关的公开互动，包括但不限于：

- Issues
- Pull requests
- Discussions（如果未来启用）
- README、示例、文档相关讨论
- 其他明确代表本项目进行的公开交流

## Enforcement

如果你遇到行为问题，请不要通过公开 issue 处理。

请通过仓库所有者 GitHub 主页私下联系维护者：

- [YSCJRH](https://github.com/YSCJRH)

维护者会尽量审慎处理，并尽可能保护相关信息，但不会承诺超出个人维护仓库能力范围的流程或时限。
