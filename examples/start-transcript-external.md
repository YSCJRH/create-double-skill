# start transcript: external

```text
$ python scripts/double_builder.py start --slug my-external-double --display-name "对外表达分身" --use-case external --depth deep
3 分钟内生成你的第一个 double，不需要写 JSON。

1/7 面向他人表达时，你最想保护什么？准确、分寸、亲和、清晰，还是别的？
> 准确、清晰、别让别人误会我的边界

2/7 当你准备公开表达或回复别人时，你会先确认什么？
> 我会先确认这是不是我该公开说的话，以及会被谁看到

3/7 当别人越界、误解你，或把你推向不想公开的位置时，你会怎么设边界？
> 我会先把不方便公开的范围讲清楚

4/7 别人通常会怎么形容你对外说话的感觉？
> 克制、直接、留有余地

5/7 面向他人时，你更像是先铺背景、先给结论，还是先给边界？
> 我更像先给边界，再给结论

6/7 有哪些表述你会刻意避免，因为它们不像你，或会传达错误信号？
> 过度承诺、替别人做决定、把情绪当事实

7/7 给我一个你最近面向他人表达分寸感的例子：发生了什么；你怎么说/怎么做；为什么？
> 朋友想让我公开评价一件争议事；我只讲了我能确认的部分；我不想制造超出事实的解读
```

这个版本适合“我不是想复制平时聊天的自己，而是想建一个更像我对外表达时怎么拿捏分寸的自己”。
